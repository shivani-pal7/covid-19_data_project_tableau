# COVID-19 Data Analysis Dashboard | Tableau Project

## Project Overview
This project presents an interactive COVID-19 dashboard created using Tableau to analyze and visualize pandemic-related data. The dashboard helps understand trends in confirmed cases, recovered cases, and deaths across different regions.

## Objective
The objective of this project is to:
- Analyze COVID-19 data
- Visualize trends and patterns
- Compare cases across regions
- Provide insights through interactive dashboards

## Dataset
The dataset includes:
- Date
- Country / Region
- Confirmed Cases
- Recovered Cases
- Death Cases
- Active Cases

Source: COVID-19 Dataset (CSV / Excel)

## Tools Used
- Tableau
- Excel / CSV
- SQL (if applicable)

## Dashboard Features
- Total Confirmed Cases KPI
- Total Recovered Cases KPI
- Total Death Cases KPI
- Recovery Analysis
- Region-wise Comparison
- Time Trend Analysis
- Interactive Filters

## Visualizations Used
- Line Chart
- Bar Chart
- Map Visualization
- KPI Cards

## Key Insights
- Identified trends in COVID-19 cases over time
- Compared recovery and death rates
- Analyzed regional differences

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Preparation
4. Dashboard Creation
5. Analysis and Insights

## Conclusion
This project demonstrates how Tableau can transform raw COVID-19 data into meaningful visual insights and support data-driven analysis.
